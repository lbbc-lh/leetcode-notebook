## BFS
时间复杂度O(n)
1. 统计入度
2. 出队（出队顺序就是拓扑排序的顺序）
3. 扩展（以下内容按顺序）
* 拓扑排序
* 判断环路
* 递推公式
* 队列类型（**优先队列**，双端队列，普通队列）
#### 509.斐波那契数(FOR)
```
class Solution {
    public int fib(int n) {
      if(n == 0 || n == 1)
      return n;
      int[]dp=new int[n+1];//定义dp数组，因为是0-n，数组长度为n+1
      dp[0]=0;
      dp[1]=1;
      for(int i = 2; i <= n; i++ )
      {
      dp[i]=dp[i-1]+dp[i-2];}
      return dp[n];
    }
}
```
